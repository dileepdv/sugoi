###########
Simple Game
###########

Simple game is a game developed in CodeIgniter. Please register to play.

*******************
Release Information
*******************

First release on Dec 11 2016

**************************
Changelog and New Features
**************************

This game has three levels. Higher the level more blocks to click and less time.

*******************
Server Requirements
*******************

PHP version 5.6 or newer is recommended.

It should work on 5.3.7 as well, but we strongly advise you NOT to run
such old versions of PHP, because of potential security and performance
issues, as well as missing features.

************
Installation
************

Download and paste the folder inside the folder where your rest of the projects are!
To run the application
